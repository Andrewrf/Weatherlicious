# Weatherlicious

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.7.

## Setup
Git Clone repository to your local machine.
Run npm install from within repository.
Run ng serve to serve app to localhost:4200.
The limited trial Accuweather key will allow 50 calls per day. If you receive a 503, this is why. 

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
